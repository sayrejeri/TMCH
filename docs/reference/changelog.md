# Changelog

This page tracks **notable updates** to the **TotalMiner Community Hub (TMCH)** documentation.

The changelog focuses on:
- new pages
- major content updates
- structural or navigation changes
- significant clarifications or corrections

Minor typo fixes and formatting tweaks are not listed.

---

## How to read this page

Each entry includes:
- a date
- a short summary of what changed
- links to affected sections when relevant

Documentation updates are continuous, so not every change appears here.

---

## 2025

### December 2025 — Initial Documentation Release

**Added**
- Getting Started overview
- Installation (initial world setup) guide
- Permissions documentation
- Creative Mode documentation
- Scripting documentation
- FAQ
- Glossary
- Contributing guide
- Discord community page

**Structural**
- Introduced sidebar sections for Getting Started, Core Systems, and Reference
- Added automatic “Last updated” metadata to all pages
- Integrated Discord as the primary community hub

**Notes**
- This marks the first complete public version of TMCH
- Content is community-driven and will expand over time
